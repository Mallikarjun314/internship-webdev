
# Border & Outline

```css
div {

  /* BORDER */
  border-width: 2px;
  border-style: solid;
  border-color: #007bff;
  border-radius: 8px;

  /* OUTLINE */
  outline-width: 3px;
  outline-style: dashed;
  outline-color: red;
  outline-offset: 4px;
  
  /* SHORTCUT */
  border: 2px solid #007bff;
  outline: 3px dashed red;
}
```
