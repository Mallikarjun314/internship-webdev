# General Layout Tags

1. `<div>   ...   </div>` (block-level element)
2. `<span>   ...   </span>` (In-line Element)
   
3. `<header>   ...  </header>`
4. `<footer>   ...  </footer>`
5. `<main>   ...   </main>`
6. `<section>   ...   </section>`

<br/>
<br/>

#### Example :

```html
<!DOCTYPE html>
<html>
    <head>
        <title>Page title</title>
    </head>
    <body>
        
        <header> ... </header>

        <main>
            <section> ... </section>
            <section> ... </section>
            <section> ... </section>
        </main>
        
        <footer> ... </footer>
        
    </body>
</html>
```

# Block-level tags vs Inline tags

<br/>


- ### Block level tags                                   
    - Starts from a new line and tags the entire width   
    - `<div>`, `<p>`, `<h1>` ... `<h6>`, `<ol>`, `<ul>` etc.

<br/>
<br/>

- ### Inline tags                                                
    - occupying only the space it needs and fitting within a line
    - `<span>`, `<a>`, `<strong>`, `<img>` etc.
